# Diagnostic Hydratation - Hydratis.app

Application de diagnostic d'hydratation personnalisé développée par Hydratis.

## Technologies utilisées

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Développement local

```sh
# Cloner le dépôt
git clone <YOUR_GIT_URL>

# Naviguer vers le projet
cd <YOUR_PROJECT_NAME>

# Installer les dépendances
npm i

# Lancer le serveur de développement
npm run dev
```

## Déploiement

L'application est déployée sur Hydratis.app.
